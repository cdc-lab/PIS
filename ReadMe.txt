PIS est un outil simplifiant l'initialisation d'un nouveau projet webSite:

- Placez le dossier "PIS" à la racine d'un disque
- Executer le script PowerShell pis.ps1 (click droit > "Executer avec PowerShell")
- Votre projet est initialisé :)

Vous pouvez modifier les fichier de base dans le repertoir "/src_files" selon vos besoins
